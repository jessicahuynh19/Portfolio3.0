GIF API

A Web based Javascript/ HTML5 Gif fetching application that uses the GIPHY API to fetch users choice of GIFs. GIFs have an click to play/ stop feature when clicked. Enter a term into the search box. As you enter terms into the search box, buttons will appear above with the term entered and gifs related to that term will appear below. If you are not sure of a term to search then feel free to select one of the already populated buttons on the page.

JLMF

Recognizing that networking events are both necessary and painful, we sought to create an app that would make attending events more pleasant and productive. This application allows the hosts of networking events the opportunity to register their events on the JLMF site. Event attendees will access the JLMF site to register for an event. At the time of the registration, the attendee will be asked to complete a short survey. The responses form the survey will be used to group the attendee by similar backgrounds and interests. This app utilizes Node, Express, MySQL, Sequelize, Express-Handlebars, Javascript, Bcrypt, Passport, and Faker.
